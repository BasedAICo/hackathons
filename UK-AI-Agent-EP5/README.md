## 📥 How to submit

1. **Fork** this repository.
2. Copy [`submissions/_TEMPLATE/`](submissions/_TEMPLATE/) to `submissions/<your-team-name>/`
   (lowercase, hyphenated, e.g. `team-rocket`).
3. Build your project in that folder and fill in its `README.md`.
4. Open a **Pull Request** into `main`. Only touch your own team folder.

Full instructions: [../CONTRIBUTING.md](../CONTRIBUTING.md).

## 🔒 Before you submit

This is a **public repo**. Never commit API keys, `.env` files, or — given the web3
sponsors — **wallet private keys or seed phrases**. Use a fresh throwaway wallet for any
on-chain work. See [../SECURITY.md](../SECURITY.md).
